# <center>Elasticsearch</center>  

Elasticsearch安装为windows服务：  
0.cmd 进入到D:\Softwares\Elastic\elasticsearch-8.6.2\bin  
1.执行elasticsearch-service.bat install  

> elasticsearch-service install	安装Elasticsearch服务  
elasticsearch-service remove	删除已安装的Elasticsearch服务  
elasticsearch-service start	启动Elasticsearch服务  
elasticsearch-service stop	停止服务  
elasticsearch-service manager	启动GUI来管理已安装的服务
